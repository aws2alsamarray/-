========================================
  إدارة ديون الإنترنت - Debt Manager
  Android App - Kotlin + Jetpack Compose
========================================

📱 وصف التطبيق:
تطبيق لإدارة ديون ومشتركي خدمة الإنترنت

✨ الميزات:
- إضافة/تعديل/حذف المشتركين
- تتبع المبالغ المدفوعة والمتبقية
- تنبيه قبل انتهاء الاشتراك (قابل للتعديل)
- 8 خلفيات بتدرجات ألوان عصرية
- الوضع الداكن/الفاتح
- اختيار العملة (دينار عراقي، دولار، يورو، ريال، درهم، دينار كويتي)
- بحث عن المشتركين
- تجديد الاشتراك بضغطة واحدة

📁 الملفات:
01_MainActivity.kt       - الشاشة الرئيسية والتنقل
02_Subscriber.kt        - نموذج بيانات المشترك
03_SubscriberRepository.kt - حفظ بيانات المشتركين
04_SettingsRepository.kt - حفظ الإعدادات
05_MainViewModel.kt     - المنطق البرمجي
06_Color.kt             - الألوان
07_Theme.kt             - السمة (فاتح/داكن)
08_HomeScreen.kt        - قائمة المشتركين
09_AddEditSubscriberScreen.kt - إضافة/تعديل مشترك
10_SettingsScreen.kt    - الإعدادات + تدرجات الألوان

🎨 الألوان: تركواز (#00897B) + ذهبي (#FFB300)

💻 كيفية الاستخدام:
انسخ الملفات إلى مشروع Android Studio في المسار:
app/src/main/java/com/agon/app/
